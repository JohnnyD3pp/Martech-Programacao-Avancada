# Martech - Programação Avançada

Este repositório reúne exercícios e exemplos desenvolvidos em **Python** durante os estudos de Programação Avançada no curso Martech.

O material aborda problemas que exigem maior estruturação lógica, incluindo programação dinâmica, manipulação de bits, uso de matrizes e organização de dados com dicionários.

## Conteúdos Abordados

- Programação dinâmica
- Manipulação de bits
- Matrizes
- Dicionários
- Algoritmo da mochila
- Maior subsequência comum
- Contagem de padrões em matriz
- Resolução de problemas computacionais

## Estrutura do Repositório

```text
.
├── Dicionário/
├── Manipulação de bits/
├── Matriz/
└── Programação Dinâmica/
```

## Descrição das Pastas

### Dicionário

Contém uma implementação do problema da mochila utilizando programação dinâmica e itens representados por dicionários.

Principais temas:

- Lista de dicionários
- Peso, valor e capacidade
- Tabela de programação dinâmica
- Recuperação dos itens escolhidos
- Maximização de valor

Arquivo principal:

- `Mochila01.py`

### Manipulação de bits

Reúne exercícios sobre representação binária e operações bit a bit.

Principais temas:

- Operador `|`
- Deslocamento de bits com `<<`
- Representação de valores binários
- Uso de `bit_length`
- Identificação de valores por combinação de bits

Arquivos:

- `Subgrade_bits.py`
- `Vinho.py`

### Matriz

Contém exercício com geração e análise de matriz binária.

Principais temas:

- Matriz com valores `0` e `1`
- Percurso por linhas e colunas
- Comparação entre pares de linhas
- Contagem de subgrades

Arquivo principal:

- `Subgrade_fb.py`

### Programação Dinâmica

Reúne exercício de programação dinâmica aplicado à comparação entre strings.

Principais temas:

- Criação de tabela dinâmica
- Comparação de caracteres
- Maior subsequência comum
- Impressão da matriz de cálculo
- Cálculo do tamanho da maior subsequência

Arquivo principal:

- `Substring.py`

## Como Executar os Arquivos

Para executar os exercícios, é necessário ter o Python instalado.

No terminal, acesse a pasta do arquivo desejado e execute:

```bash
python nome_do_arquivo.py
```

Exemplo:

```bash
python Mochila01.py
```

Alguns arquivos solicitam entrada pelo teclado; outros executam com valores definidos diretamente no código.
